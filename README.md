<h2>What is Flint?</h2>
A website you can use to unrestrict internet by connecting to servers around the world! Simply enter your wisp server or use one provided and boom! You can now browse the internet.
<h2>How does it work?</h2>
Flint is a modified version of Scramjet by MercuryWorkshop. Huge shoutout to them.
